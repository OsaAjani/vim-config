# Config vim editor
Config for vim.

**List of configs :**
Tab autocomplete
PHP optimized
Auto indent
4 spaces indent
Probably some other things i've forgot

## How to use
Copy content of the repo to ```$HOME/```
Run ``` git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim```
Then run ```vim +PluginInstall +qall```
